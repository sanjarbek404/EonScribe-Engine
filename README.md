# 📜 EonScribe Engine

> **The Infinite Ledger of Your Agency.**

EonScribe is a next-generation "Choose Your Own Adventure" (CYOA) engine that leverages advanced AI to provide a truly infinite narrative experience. Unlike traditional games with pre-set branches, EonScribe weaves a unique story in real-time based on your specific actions, ensuring no two journeys are ever the same.

![EonScribe Engine](https://picsum.photos/seed/cyberpunk/1200/400)

## ✨ Features

- **🧠 Infinite Narrative Intelligence**: Powered by `gemini-3.1-pro-preview`, the engine narrates a high-stakes adventure where every choice shifts the fabric of the multiverse.
- **🎨 Visual Continuity**: Real-time image generation using `gemini-3-pro-image-preview`. The AI maintains a consistent visual style and character description so your story looks uniform from start to finish.
- **🎒 Dynamic Journaling**: An automated sidebar that tracks your **Current Quest**, **Location**, and **Inventory**. The AI identifies new items or objectives within the narrative and updates your journal instantly.
- **🏺 The Sage's Presence**: A floating AI guide available for consultation. Ask the Sage for lore insights, hints, or just context on your current predicament.
- **🌑 Elegant Dark UI**: A bespoke, sophisticated design theme featuring gold accents, Georgia serif typography, and cinematic scanline effects.
- **⚙️ Ritual Quality Control**: Configure the intensity of your visual visions with adjustable image resolutions (1K, 2K, 4K).

## 🚀 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- A Google Gemini API Key (Set in your environment or Secrets panel)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/eonscribe-engine.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 📜 How it Works

The engine operates on a sophisticated loop:
1. **The Game Master**: Processes your history and choice to generate a structured JSON containing the next narrative block and updated state.
2. **The Visionary**: Takes the `imagePrompt` generated by the Game Master and renders a cinematic visual representing the scene.
3. **The Chronicler**: Updates the local game state, ensuring your inventory and quest progress are persisted in your browser's local storage.
